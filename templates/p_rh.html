Projeto RH - Análise de Dados

Primeiramente começamos com uma visualização geral das primeiras 5 linhas da base de dados e em seguida renomeamos as colunas para o português.

Verficamos agora se há campos nulos na base de dados.

E em seguida chamamos as principais características estatísticas da base de dados para termos uma noção das informações gerais possuídas.

Vemos que o mínimo de experiência que temos nessa base de dados é de 1 ano aproximadamente, com um salário de 37.731,00 dólares, e uma máxima de 10 anos aproximadamente com um salário de 122.391,00 dólares.

Logo abaixo podemos ver pelo gráfico de densidade que há dois grupos de concetração de dados que coincide com a Renda e o tempo de Experiência.

Depois plotamos um gráfico boxplot para visualizarmos melhor a distribuição dos dados na coluna de Renda e Xp e podermos analisar as medianas e os quartis..

Aqui plotamos um grádico de dispersão para analisar o comportamento de dados, e podemos ver que há uma linha de regressão linear presente, apontando que a medida que o tempo de experiência cresce, o sálario também aumenta.

Por final fizemos dois tipos de gráfico de correlação para analisar se há uma correlação forte entre os dados da coluna Renda com a coluna Xp atráves dos gráficos de correlação Pearson e Spearman.

Por fim podemos concluir e responder a pergunta que é evidente que o tempo de experiência de um profissional influencia no seu salário anual.
